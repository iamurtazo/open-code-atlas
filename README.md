# Open Code Atlas

Open Code Atlas is an open-source platform for learning programming through curated YouTube courses.  
It transforms freely available video playlists into structured, enrollable courses with clear learning paths.

Users can:
- Discover programming courses by language
- Enroll and manage their learning dashboard
- Learn from existing high-quality content without paywalls

The platform is built with a modular architecture to support both web users and API consumers.

## Core Principles
- Free education
- Open-source collaboration
- Practical, real-world learning

Contributions, feedback, and ideas are welcome.
